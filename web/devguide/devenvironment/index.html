Setup of Development Environment

Development Environment

In order to create your own Impresario Macro libraries or to build Impresario yourself from its source code, you need a development enviroment.

In general three components are required:

  • Qt Framework 5.5 with QtCreator
  • A suitable compiler for QtCreator
  • Git as version control system

The following two subsections show how to set it up on Windows and Linux systems.

Setup for Windows systems

On Windows systems Impresario currently supports the MinGW C++ compiler 4.9.2 (GNU gcc 32bit) and Microsoft’s C++ compiler version 18.0 with runtime 12.0 in 32bit and 64bit. The Microsoft compiler for example is shipped with Visual Studio 2013.

  • Go to http://www.qt.io/download-open-source/ and download the Qt Online Installer for Windows.
  • Run the downloaded installer. After creating a Qt account (it’s free) you can choose which Qt components to install. The following screenshot shows what is needed by Impresario: Color legend:
    • Green: Mandatory components to be installed in any case
    • Yellow: To be installed if you like to use MinGW compiler
    • Blue: To be installed if you like to use Microsoft’s C++ compiler
  • If you choose one of the msvc2013 packages during installation of Qt, you have to install the Microsoft compiler as well. The easiest way is to install MS Visual Studio 2013 Express for Desktop. You can download it from https://www.visualstudio.com/downloads/download-visual-studio-vs (it’s free).
  • Download Git for Windows from https://git-scm.com/download/win. Afterwards, run the installer.

Setup for Linux systems

On Linux systems the GNU 64bit gcc is the compiler of choice.

  • Go to http://www.qt.io/download-open-source/ and download the Qt Online Installer for Linux.
  • Run the downloaded installer. After creating a Qt account (it’s free) you can choose which Qt components to install. The following screenshot shows what is needed by Impresario:
  • If not already installed on your system you have to install gcc itself. On Debian based systems like Ubuntu or Linux Mint with apt repository maintenance this is achieved by running sudo apt-get install build-essential from the command line.
  • Install Git on your system if not already done. For Debian based systems run sudo apt-get install git from the command line. The website https://git-scm.com/download/linux shows how to install Git on other Linux systems.
