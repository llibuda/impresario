System requirements

To run the binaries offered for download here the following requirements must be met:

  • Windows 64bit operating system or Linux 64bit operating system

  • Multicore processor

  • At least 2GB RAM

On Windows Impresario was tested with Windows 7, Windows 8, Windows 8.1, and Windows 10.

On Linux Impresario was tested on Ubuntu 14.04 LTS and Linux Mint 17, so at least Debian based systems are expected to work.

Impresario can also be built as 32bit application. However, only 64bit binaries are offered here.
