User's Guide overview

The User's Guide contains information on how to operate Impresario in daily use.

It gives a detailed introduction in its purposes, explains how to install and configure Impresario, and finally provides information about how to use all built-in features in the most efficient way.

The following chapters are available:

  1. Introduction

    Detailed introduction to Impresario, its purpose, basic concepts, and features. It also contains a short history of Impresario.

  2. Installing Impresario

    All about the installation and deinstallation of Impresario on Windows and Linux systems.

  3. Introducing Impresario's GUI

    When you start Impresario for the first time, everything is new. Here you find a short introduction to the most important components of Impresario's user interface.

  4. Editing process graphs

    Creating and modifying process graphs is the most frequent task. This section documents how you can edit them.

  5. Running process graphs

    Running process graphs and visualizing results is what Impresario was made for. Here you find detailed descriptions how you can run a process graph and what is displayed while running a process graph.

  6. Reference: Macro database window

    The Macro database window is essential for creating process graphs. This section documents the window's behavior and features.

  7. Reference: Property window

    The property window allows to change settings for the process graph and macro parameters. It also provides detailed information about every component in a process graph.

  8. Reference: Settings

    Impresario can be configured to suit your needs. This reference covers all available settings.

  9. Reference: Commands

    Reference and brief description of all available commands.

  10. Reference: Auxiliary docking windows

    This section documents Impresario's auxiliary docking windows. They are not necessarily required for work but may provide additional and more detailed information in some situations.

  11. Release history

    Release history of Impresario
