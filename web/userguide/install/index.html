Installation

Impresario binaries are distributed in compressed archives, “zip” archives on Windows platforms and “tar.gz” archives on Linux platforms. Impresario does not make any changes to the operating system, it runs locally from the folder you installed it to.

Installation on Windows

  • Download the two zip archives Impresario-win64-X.Y.Z.zip and ImpresarioMacros-win64-X.Y.Z.zip respectively, where X, Y, and Z denote the major, minor, and patch version of the packages.
  • Create a new folder somewhere on your hard drive where you would like Impresario to be installed to, e.g. D:\Impresario-2.0.0.
  • Call your favorite zip handling tool to uncompress both archives into the just created folder.
    Important: Uncompress Impresario-win64-X.Y.Z.zip first! During uncompression of ImpresarioMacros-win64-X.Y.Z.zip, the zip tool may prompt you to overwrite the file bin\Impresario.xml. Please let it overwrite the file.
  • You are done. You can now start Impresario by running Impresario.exe in the bin subdirectory.

Installation on Linux

  • Download the two compressed tar archives Impresario-linux64-X.Y.Z.tar.gz and ImpresarioMacros-linux64-X.Y.Z.tar.gz respectively, where X, Y, and Z denote the major, minor, and patch version of the packages.
  • Create a new folder somewhere on your hard drive where you would like Impresario to be installed to, e.g. ~/Impresario-2.0.0.

  • Move both tar.gz archives into the just created folder and then run the following commands from within this folder in the given order:

    tar xzvf Impresario-linux64-X.Y.Z.tar.gz
tar xzvf ImpresarioMacros-linux64-X.Y.Z.tar.gz
# optional clean up
rm *.tar.gz

  • You are done. You can now start Impresario by running ./impresario.sh in the bin subdirectory.
    Important: Do not run the program ./Impresario! This will run Impresario with an incorrect setup of LD_LIBRARY_PATH.

Uninstall Impresario

To uninstall Impresario just delete the folder containing Impresario.

Impresario stores its user specific settings on Windows in <PATH_TO_YOUR_USER_DIRECTORY>\AppData\Roaming\Impresario. On Linux these settings are stored in ~/.config/Impresario.

You may like to delete these settings as well.
